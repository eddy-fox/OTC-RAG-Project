# OTC Drug Document-based RAG System

## Overview
### 프로젝트 개요
OTC 의약품 첨부문서를 검색 근거로 활용하는 RAG 기반 QA 백엔드 프로젝트입니다.
질문에서 약물명을 식별하고 관련 문서를 검색한 뒤, 검색 결과를 OpenAI API에 전달해 근거 기반 응답을 생성하도록 구성했습니다.
RAG 로직은 별도 모듈로 분리해 프론트엔드와 연동 가능한 구조로 설계했습니다
