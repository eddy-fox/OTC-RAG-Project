# OTC Drug Document-based RAG System

## Overview
### 프로젝트 개요
일반의약품(OTC) 관련 질문은 단순 제품 검색만으로 해결되지 않는 경우가 많다. 사용자는 특정 약의 복용 가능 여부, 주의사항, 병용 여부처럼 문맥이 필요한 질문을 던지지만, 일반 검색은 필요한 정보를 바로 찾기 어렵고 범용 LLM은 근거 없는 답변을 생성할 수 있다.
이 프로젝트는 이러한 문제를 해결하기 위해 의약품 첨부문서를 근거로 검색하고 답변하는 RAG 기반 QA 시스템을 구축한 프로젝트이다. 사용자의 질문에서 약물명을 식별하고, 관련 첨부문서를 검색한 뒤, 검색 결과를 바탕으로 응답을 생성하도록 구성하였다.
또한 실제 사용성을 높이기 위해 약물명 후보 선택, 문서 청킹, 메타데이터 기반 검색, 근거 중심 프롬프트 설계를 포함해 전체 질의응답 흐름을 하나의 파이프라인으로 구현하였다.
